Joe Moceri

I'm a Full Stack Software Developer. I started writing code in 2011 using C first then Python, which was much easier to work with. Using the Python Turtle I drew my first graphic on screen, and I was hooked on coding ever since. Software has always fascinated me, I wanted to write code for video games and eventually I learned how to do this first using Processing (sketches here). Around this time I started learning JavaScript which paired well with Processing, and while Processing is great, I needed something more feature dense and suited for games, so I picked up Unity3D, where I started with C#. I built a series of games like Scrap Metal, a first person multiplayer shooter with a team of 5 in college, and more (they're available on GitHub). From there I began my career as a Full Stack Web Developer, where I worked with databases such as SQL Server, MySQL and PostgreSQL, writing SQL queries, stored procedures, and more that were part of many 3rd party integrations. I did a lot of database schema design, management, and DevOps working with a small team. Things like setting up Build Pipelines on Visual Studio Team Services, automating tasks using powershell, deployment to virtual machines, participating in pull request peer review and mentorship, IT support, QA, unit and integration tests, management and watching of multiple applications using homebrew software custom written to deliver and receive updates (push systems). Many of these systems were integrated into a SaaS platform built for the business to manage their clients and products. The platform was written in AngularJS with a .NET MVC 5 Web API Backend using OAuth authentication. I was part of a team that migrated and delivered this platform using an agile methodology in bi-weekly sprints. The team and business decided to migrate to the cloud, using both Amazon and Microsoft Azure to host and run software, including the following products for Amazon: Amazon Web Services, EC2, their service for message queues (SQS), Simple Email Service, Elastic Beanstalk, RDS, and Lambda; and the following products for Microsoft: Microsoft Azure, Key Vault and WebJobs. In my career and personal projects moving forward I've also used these technologies, and applied and built upon these ideas. Recently, I've spent time working with WebRTC and multiple integrations with 3rd-party providers for video chat, the ability to text phones using a web browser, and other real-time communications using .NET and Angular. I have a huge passion for game design and design in general and so in my spare time I love to work on indie video game development.
